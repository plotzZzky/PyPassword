### feito com Django, PostgresSql e Pykeepass.

- [x] Cria um arquivo .kdbx com senha;
- [x] Permite criar, editar e deletar senhas;
- [x] Sistema de login;
- [ ] melhorias em breve...


### Importante:
- É necessario um servidor PostgreSql funcionando
